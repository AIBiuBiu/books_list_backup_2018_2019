\begin{frame}

\begin{itemize}
	\item 参数配置
	\small
	
	      Aomi自创立以来，除了对功能要求很高以外，对性能要求也越来越高。
      Aomi性能测试在不断向前发展，横向、纵向都在不断深入、拓宽，不断提出更高要求。
      
      
      一个应用

      的广度，这是一个挑战。专业的测试需要专业的格式，	  
	    	
\end{itemize}
